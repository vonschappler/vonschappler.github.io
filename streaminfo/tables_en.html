Legend

  • Cost makes reference to how many vsCoins are spend for using the command
  • Cooldown makes reference to how frequently the command can be used
  • Affected tells if the command is affected by localization settings
    • The command response is affected by localization settings
    The command response is not affected by localization settings and its response will be returned only in EN-US or in both EN-US and PT-BR, depending on the command
  • Using -w (as displayed in some commands) will return the command response on the viewer whisper, IF their account settings allow recieving whispers from non-friend accounts
  • All commands in this list can be used directly in vonschappler_bot whisper. Just type /w vonschappler_bot (command), replacing (command) for the command you want to send
